# 🍕 End-to-End Pizza Sales SQL Analysis Project

## 📌 Project Overview

This project focuses on analyzing Pizza Sales data using SQL to uncover valuable business insights related to sales performance, customer behavior, product popularity, and revenue generation.

The analysis was performed using MySQL and demonstrates practical SQL skills commonly required for Data Analyst and Business Analyst roles.

---

## 🎯 Objectives

The primary objectives of this project were:

* Analyze overall sales performance
* Calculate total revenue generated
* Identify top-selling pizza types
* Understand customer ordering patterns
* Analyze category-wise sales distribution
* Evaluate revenue contribution by products
* Generate actionable business insights

---

## 📂 Dataset Information

The project uses four relational tables:

### Orders

Contains order date and time information.

### Order Details

Contains quantity and pizza information for each order.

### Pizzas

Contains pizza sizes and pricing information.

### Pizza Types

Contains pizza names, categories, and ingredients.

## 🔍 Business Questions Solved

### Basic Analysis

1. Retrieve the total number of orders placed.
2. Calculate the total revenue generated from pizza sales.
3. Identify the highest-priced pizza.
4. Determine the most common pizza size ordered.
5. Find the top 5 most ordered pizza types.

### Intermediate Analysis

6. Calculate total quantity sold by pizza category.
7. Analyze order distribution by hour.
8. Determine category-wise pizza distribution.
9. Calculate average pizzas ordered per day.
10. Find the top 3 pizza types based on revenue.

### Advanced Analysis

11. Calculate percentage revenue contribution of each pizza type.
12. Analyze cumulative revenue over time.
13. Identify top 3 revenue-generating pizzas within each category.

---

## 📈 Key Findings

### Sales Performance

* Total Orders: 21,350+
* Total Revenue: $817,860+
* Average Daily Pizza Sales: 138+ pizzas

### Customer Preferences

* Large (L) pizzas were the most frequently ordered.
* Classic category pizzas had the highest sales volume.
* Chicken-based pizzas generated the highest revenue.

### Top Revenue Generators

1. Thai Chicken Pizza
2. Barbecue Chicken Pizza
3. California Chicken Pizza

### Pricing Analysis

* The Greek Pizza (XXL) was the highest-priced pizza on the menu.

### Customer Ordering Behavior

* Peak order volume occurred during lunch and evening hours.
* Demand remained consistent throughout the sales period.
